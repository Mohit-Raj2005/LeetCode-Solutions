# LeetCode Solutions - JavaScript

This folder contains my solutions to LeetCode problems implemented in JavaScript.

## Structure

Each problem solution is organized in its own file, typically named after the problem number and title (e.g., `001-two-sum.js`).

## How to Run

To run a solution, you can use Node.js:

```bash
node 001-two-sum.js
```

Make sure you have Node.js installed on your system.

## Notes

- Solutions are written in modern JavaScript (ES6+).
- Focus is on clarity and efficiency.
- Comments are included to explain the approach and time/space complexity where relevant.
